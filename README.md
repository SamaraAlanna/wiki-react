# Wiki React

Projeto desenvolvido como parte da formação em React da [DIO (Digital Innovation One)](https://www.dio.me/), utilizando React com componentes funcionais, hooks e consumo de API externa.

## Funcionalidades

- Busca de perfil pelo nome de usuário do GitHub
- Exibição de avatar, bio, seguidores e repositórios públicos
- Listagem de repositórios com linguagem, estrelas e forks
- Navegação entre páginas com React Router DOM
- Tratamento de usuário não encontrado

## Tecnologias

- React 18
- React Router DOM
- API REST do GitHub
- CSS puro
- Create React App

## Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/SamaraAlanna/wiki-react.git

# Entre na pasta
cd wiki-react

# Instale as dependências
npm install

# Rode o projeto
npm start
```

Acesse [http://localhost:3000](http://localhost:3000) no navegador.
